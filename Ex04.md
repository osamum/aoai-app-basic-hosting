# 演習 4 :  Azure サービス間の認証設定

現在の演習用アプリケーションは、自身をホストしている Azure App Service 以外に Azure OpenAI と Azure AI Search を使用しています。

それらのサービスに接続する際の認証情報である API キーは、ローカルで開発する際は環境変数の設定ファイルである `.env` に、Azure App Service では、環境変数の設定 UI に平文で記述しています。この方法は簡単に実装できる一方で、API キーが漏洩するリスクがあるため、運用環境では推奨されません。

この演習 4 では、より安全に Azure サービス間の認証を行うための以下の二つの方法について紹介します。

1. [演習 4-1 : Key Vault へのキーの登録と利用](Ex04-1.md)
2. [演習 4-2 : マネージド ID によるサービス間認証](Ex04-2.md)
