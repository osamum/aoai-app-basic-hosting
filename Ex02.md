# 演習 2 : アプリケーションをホストするための基本的な設定

この演習では、[演習 1](Ex01.md) で演習用アプリケーションをデプロイした Azure App Servive(Web App) の現在の設定の確認と Web サイトとして運用するため基本的な設定を行います。

この演習では、一般的な Web サイトでコンテンツをホストする際に必要となる以下の基本的な内容について手順を実施します。ただし、ドメインの購入など実施が難しいものについては手順の説明のみを行います。

1. ホスティングに関する設定の確認
2. App Service ログの設定と有効化
3. バックアップ
4. カスタム ドメインの設定
5. 認証の追加
